# Roadmap

## Phase 0
Lock PRD, domain boundaries, anti-cheat policy, and event schema.

## Phase 1
Bootstrap monorepo, shared tooling, local infrastructure, and CI.

## Phase 2
Implement auth, profile, and role model.

## Phase 3
Implement matchmaking queue and battle room orchestration.

## Phase 4
Implement scoring and hidden MMR plus visible tiers.

## Phase 5
Implement question delivery, calibration, and review metadata.

## Phase 6
Implement anti-cheat detection and moderator workflows.

## Phase 7
Implement weakness graph, mistake notebook, and retention loops.

## Phase 8
Closed beta, load hardening, and SLO stabilization.

## Phase 9
Public launch with season operations.
