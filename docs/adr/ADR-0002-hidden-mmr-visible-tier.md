# ADR-0002: Hidden MMR + Visible Tier Ladder

Date: 2026-03-19
Status: Accepted

## Decision
Use hidden MMR for matchmaking and visible tier divisions for progression UX.

## Rationale
- Preserves match quality and fairness.
- Reduces ladder anxiety from volatile rating values.

## Consequences
- Need transparent player communication on rank behavior.
- Requires robust season reset and placement logic.
